# Telegram Leads Parser

Userbot на Telethon: мониторит фриланс-чаты, находит заявки
по триггер-словам, скорит их и отправляет в канал + Битрикс24.
Работает на VPS 24/7.

## Как это работает

Telegram-чаты → фильтр по ключевым словам → скоринг заявки
→ карточка лида в канал (с черновиком ответа) → лид в Битрикс24 (REST API)

## Скриншот

(сюда — твой скрин с карточкой лида, замажь имена и названия чатов)

## Стек

Python · Telethon · Bitrix24 REST API · VPS (Linux)

## Запуск

1. `pip install -r requirements.txt`
2. Скопировать `.env.example` → `.env`, заполнить ключи
3. Указать свои чаты и триггер-слова в `config.py`
4. `python -m parser`
